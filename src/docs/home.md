## Home

Welcome to the official documentation for **Pogly** - the free, real-time collaborative stream overlay editor. Think of it like a shared design canvas for your OBS overlay: multiple people can add, edit, and arrange elements together, and everything updates live on your stream.

These docs cover everything from your first overlay to advanced widget development and the HTTP API.

### New to Pogly?

Start with the **Getting Started** section - it walks you through creating an overlay and adding it to OBS in about a minute. From there, **Elements**, **Media Players**, and **Layouts** cover the day-to-day basics.

### How these docs are organized

The sidebar groups pages by who they're for:

- **For everyone** - the core concepts every user needs: elements, media, layouts, collaboration, and permissions.
- **Editor** - getting the most out of the editor: canvas tools, keyboard shortcuts, chat, settings, assets, and backups.
- **Advanced** - developer topics: building widgets and using the HTTP API.
- **Help** - troubleshooting, FAQ, and a glossary of terms.

### Finding what you need

- **Search** - use the search bar at the top to jump straight to any topic.
- **On this page** - when a page is open, its sections appear under it in the sidebar so you can jump to a specific spot.
- **Links** - the bottom of the sidebar has links to our GitHub and Discord.
