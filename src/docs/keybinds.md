## Keyboard Shortcuts

Pogly is built to be driven quickly from the keyboard. This page is the complete reference for every shortcut, grouped the same way they appear in settings.

> [!TIP]
>
> On Mac, shortcuts that use **Ctrl** below use **Cmd** instead.

### Customizing Shortcuts

Every shortcut can be remapped in **Settings → Keybinds**:

- Click any shortcut and press the new key combination to record it
- Conflicting bindings are highlighted in orange so you can spot clashes
- Reset any individual shortcut back to its default at any time

### History

| Action | Default |
|---|---|
| Undo | Ctrl+Z |
| Redo | Ctrl+Shift+Z |
| Redo (alternate) | Ctrl+Y |

### Elements

| Action | Default |
|---|---|
| Delete element | Delete |
| Copy | Ctrl+C |
| Paste | Ctrl+V |
| Cut | Ctrl+X |
| Duplicate | Ctrl+D |

> [!NOTE]
>
> Paste is context-aware: an image URL becomes an image, a media link becomes a media player, a copied element becomes a duplicate, and plain text becomes a text element. See the Elements section.

### Visibility

| Action | Default |
|---|---|
| Show element (opacity 100%) | Page Up |
| Hide element (opacity 0%) | Page Down |

Hiding and showing can play a smooth fade instead of an instant change - configure it under Settings → General.

### Transform

| Action | Default |
|---|---|
| Flip horizontal | Ctrl+F |
| Flip vertical | Shift+F |
| Enable warp mode | Shift+W |
| Enable clip mode | Shift+C |
| Enable scale mode | Shift+S |
| Reset transforms | Shift+R |
| Lock drag axis (hold) | Shift |

> [!NOTE]
>
> Hold **Shift** while resizing to keep the original aspect ratio. If you'd rather aspect-lock be the default and Shift free it, enable **reverse aspect ratio lock** in Settings → General.

### Nudge

Move the selected element one pixel at a time, or ten pixels with Shift. Only one direction moves at a time.

| Action | Default |
|---|---|
| Nudge | Arrow keys |
| Nudge (large) | Shift + Arrow keys |

### Canvas

| Action | Default |
|---|---|
| Center view on stream preview | Home |
| Pan up | W |
| Pan down | S |
| Pan left | A |
| Pan right | D |

### Spotlight

| Action | Default |
|---|---|
| Open Spotlight search | Ctrl+Space |
| Open Spotlight (alternate) | Ctrl+K |

Spotlight is the fastest way to add an asset - type to search your images, emotes, and widgets, then click a result to drop it on the canvas.
